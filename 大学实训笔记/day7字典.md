# 字典（dict）

~~~markdown
映射关系：一一对应

1. 是Python中唯一的映射类型
2. 结构：
		key：键
		value：值
		k-v:键值关系 ---键值对：
			k-v,key-value,entry	
~~~

* 创建

~~~markdown
1. 手工创建	
		d = {'one':1,'two':2...}
		变量 = {key1：value1，key2：value2...}
2. 构造方法
		dict()：
			返回一个新的空字典
		dict(mapping):
			接收一个映射类型的对象，并将该映射类型元素转换成字典
		dict(iterable):
			将可迭代对象中的元素打包成字典
			可迭代对象必须是具有映射关系的可迭代对象（每个大元素中必须有两个小元素）
			例如：[(1,2),(3,4),(5,6)]
			     (['one',1],['two',2])
			     ['ab','cd']
		dict(**kwargs):
			使用关键字参数传值，并将关键字参数打包成字典中的元素
			关键字作为键，值作为字典的值
~~~

* 访问

~~~markdown
1. 访问一个
		d = {'one':1,'two':2}
		d[’one‘] # 1
2. 访问全部
		遍历
		直接遍历只能遍历出字典的键
		遍历键，遍历值，遍历键值对
~~~

~~~python
for i in d.keys():  # 遍历键
    print(i)

for i in d.values(): # 遍历值
    print(i)

for i in d.items(): #遍历键值对
    print(i)
~~~

* 添加

~~~markdown
1. 手动添加
		d = {'one':1,'two':2}
		d['three']=3
		给字典中不存在的键赋值
~~~

* 修改

~~~markdown
1. 手动修改
		d = {'one':1,'two':2}
		d['one']=99
		语法：将已经存在的键的值进行重新赋值
~~~

* 删除

~~~markdown
1. clear()：
		清空字典
2. pop(k[,d])：
		删除键为k的键值对，如果k不存在，但是d给出，则返回d值，如果d没设置，则抛出KeyError异常
		如果k存在则删除对应的键值对，并返回k对应的值
3. popitem()：
		从字典中随机删除一组键值对，并返回该键值对
		形式为： （key，value）
		如果字典为空，则抛出异常：KeyError
~~~

* 其他方法

~~~markdown
1. copy()：
		浅拷贝
2. fromkeys(iterable, value=None)
		将可迭代对象的每个元素，作为键，将value作为每个键的值，打包成一个新的字典
3. get(k[,d])：
		获得一个值
		如果k这个键在字典中，则返回该键对应的值
		如果不存在，则返回d值
4. setdefault(k[,d])：
		如果k这个键在字典中，则返回该键对应的值
		如果不存在，则在字典中添加一个键值对：键为k，值为d
5. update([E, ]**F)
		使用参数中的键值对，更新原有的字典
~~~

* 字典的特点

~~~markdown
1. 不支持切片操作
2. 不是序列
3. 是无序的
4. 键是唯一的不可重复的
5. 是一个可迭代对象
~~~

* 字典的要求

~~~markdown
1. 字典中的键：
		必须是可哈希的
		必须是不可变类型
2. 元组是不可变类型，可以作为字典的键
		但是，必须保证元组中的任何元素，也都是不可变类型
		（如果元组中有可变类型对象，则会报错）
~~~
