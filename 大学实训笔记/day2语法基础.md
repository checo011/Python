 # Python语法基础

* 输出函数

~~~markdown
1. print(参数)   
		print: 函数名
		():调用   
2. print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
		*obj：可变长参数   （任何参数中只要有object作为参数，说明可以传递任何类型的数据）
				可以传递任意多个且任意类型的对象
				多个参数用逗号隔开
		sep: 分隔符  默认是空格（默认参数： 即使不给它传参数，也可以使用等号后面的默认参数）
		end: 结束符  默认是换行 
		file: 流对象
		flush：是否清空缓存
		
3. print函数也称之为输出函数  向控制台输出指定内容
~~~

* 输入函数

~~~markdown
1. input(参数)
		从控制台等待用户输入，并将用户输入的数据返回给程序
2. input([prompt])
		prompt： 是用方括号抱起来的，可选参数（可以传，也可以不传）
  				 是提示内容
  				 可以是任何类型 
        input可以返回一组人输入的数据，这组数据一定是字符串类型      
~~~

* 变量

~~~markdown
1. Python中的变量，变量中存储的不是数据本身，而是数据的首地址
2. Python中的变量相当于一个标签，可以从一个对象上撕下来，重新粘贴到新的对象上
		变量本身没变，内部的首地址发生改变
3. 变量要先赋值后使用
4. 变量要符合标识符命名规范
		重要
~~~

* 标识符命名规范

~~~markdown
1. 语法（必须遵守）
		1. 使用合法标识符：字母，数字（数字不能开头）,下划线
			看情况：
				Python2.x : 不行 --- 
					默认使用的编码：ASCII 不包含中文，会报错
				Python3.x ：可以 ---
					默认使用的编码：unicode：UTF-8 
					unicode：万国码，包含中文
			注意：强烈不建议使用中文作为标识符，因为可能出现中文乱码问题			
		2. 大小写敏感
        	区分大小写
		3. 不能使用关键字，和保留字
			关键字：Python已经占用的名称，并且，该名称有着特殊的语法含义，且不能更改或赋值。
			if  while for else with except try def class from  import 
			保留字：Python已经占用的名称，并已经被赋予功能，但是可以进行修改或赋值，不建议修改：因为导致Python中原有的功能失效
			数据类型，都是保留字，print，input，type 
		4. 没有长度限制
2. 开发习惯(应该尽量遵守，不会影响语法)
		1. 望文生义
			1. 让信息密度更大
			2. 准确表达
		2. 大小写：
			1. 常量：全大写
			2. 包名：全小写
			3. 类名：大驼峰（首字母大写，其他单词首字母也大写）
			4. 函数名，变量名，模块名：小驼峰（首字母小写，其他单词首字母大写）
			5. 其他命名：
				所有的单词都小写，并用下划线连接
~~~

* 数据类型

~~~markdown
1. 整型： int 
		整数，1,2,3
		Python2.x int long
		Python3.x int 
2. 浮点型：float
		小数： 1.5 
		科学计数法：1e+10
		扩展：
		1.5 ---  1.  +  .5 
		浮点数： 是由定点整数 和 定点小数构成
		浮点数的实现：slot（槽）容量是4个字节
		浮点数需要占用2个slot 
		
3. 字符串：str
		1. 单引号字符串
			'abc'    ： Python字符串的通用形式
		2. 双引号字符串
			“abc”    ： 用于引号的嵌套
		3. 三引号字符串
			"""abc"""  ：长字符串，可以换行，用于做文档注释
			'''abc'''  
4. 布尔： bool
		有且只有两个字面值： 
		True 
		False
		1. 任何非零数据都可以代表True
		2. 任何零值都可以代表False
			0  '' [] {} None 
		3. True可以代表1 
		4. False可以代表0
		为什么数字可以和bool类型进行计算，bool类型的父类是int（bool类型是int类型的子类）
		while 1 和 while True 谁的执行效率更高？
		看情况：
			1. Python2.x
				while 1 的效率高
				1. 数字1是个数，不可以被修改，无需校验
				2. True是一个保留字，在循环之前要进行校验，查看True中是否还是True
				
			2. Python3.x
				一样高
				True是一个关键字，不可以被修改，无需校验。
5. 空值：None
6. 复杂类型：
		列表：list  [1,2,3]
		字典：dict  {'one':1,'two':2}
		集合：set   {1,2,3}
		元组：tuple (1,2,3)
~~~

* 运算符

~~~markdown
表达式： 
字面值或变量和运算符拼接组合的式子
1+1
a=1
b=2
a+1
a+b

1. 常用运算符：
		+ - * /(真除法) %  //（地板除-向下取整 **幂运算
2. 赋值预算：
		= 
		a=b  b给a赋值
		增强赋值运算：
		+=   a+=1   a=a+1   (Python中没有++操作)
		-= *= /= //= %= **=
3. 布尔运算（比较运算符）：
		> < >= <= == != 
		Python 支持连等
4. 逻辑运算符
		and  or  not 	
5. 位运算：（相对于2进制来说）
		>> 右移
		<< 左移  a<<n  a*2^n
6. 三元运算符：
		a = (值1 if 布尔表达式 else 值2)
        先判断布尔表达式，如果未真则返回值1，否则返回值2
~~~
