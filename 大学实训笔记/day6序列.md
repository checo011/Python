# 序列

* 各种序列类型及特点

~~~markdown
1. int ： 整型
2. str ： 字符串
3. bool ： 布尔类型
4. float： 浮点型
5. None：空值
6. list：列表
7. tuple：元组
8. range:range类型

特点：
1. str，list，tuple，range
		1. 支持下标操作（索引操作）
		2. 支持切片操作
		3. 都是可迭代对象
		4. 都可以存储多个数据
		5. str，list，tuple 可以存储多种数据
~~~

* 可迭代对象（iterable）

~~~markdown
1. 概念：
		拥有迭代器的对象，称之为可迭代对象
2. 如何获取迭代器：
		iter(可迭代对象)
		该方法会返回指定对象的迭代器对象

		可迭代对象中包含了迭代器对象
3. 常见的可迭代对象：
		str，list，tuple，range
4. 最简单的验证可迭代对象的方法：
		使用for循环
~~~

* 迭代器（iterator）

~~~markdown
1. 概念：实现了__iter__() 和 __next__() 的对象，都是迭代器
		体现为：迭代器可以使用于 iter()  next()
2. 迭代器也是一个可迭代对象
3. 迭代器的迭代器是自身
4. next(迭代器)用于获取下一个元素
5. 迭代器的底层结构：
		1. 结构类似于数组
		2. 在原始状态，第一个数据之前有一个游标
		3. 每调用一次next()，则游标向后移动一个单位，并将划过的数据进行返回
6. 迭代器中的数据，原则上和可迭代对象中的一致
		1. 数据一样的原理：迭代器本质上是将可迭代对象中的数据拷贝出了一份
		2. 数据不一致的情况：如果发生了：列表的数据发生更改，此时迭代器尚未来得及拷贝数据时
		3. 如果游标已经到最后，再次调用next() 会报错： StopIteration
7. 可迭代对象不一定是迭代器，迭代器一定是可迭代对象
8. iter()函数的底层实现类似于：__iter__()    next()函数的实现底层类似于：__next__()
9. for 循环的底层实现
		1. 对可迭代对象调用iter(),获取迭代器
		2. 对迭代器循环调用next()，获取数据
		3. for循环检测StopIteration，如果捕获异常，则停止循环（for循环处理了，迭代器中的异常）
~~~

* 序列相关的函数

~~~markdown
1. max()：最大值
		max(iterable, *[, default=obj, key=func])
				返回一个最大值
				1. 可以接受一个可迭代对象，返回该可迭代对象的最大值
				2. 可以接受多个可迭代对象，返回最大的可迭代对象
				3. 注意比较的数据需要是同一个类型
    	max(arg1, arg2, *args, *[, key=func])
    			可以接收多个值，并返回最大值
2. min()：最小值
		同上，求最小
3. sum()：求和
		sum(iterable, start=0)
			将可迭代对象的每个元素，进行相加求和
			start：起始值默认是0，可以修改
			
4. zip()：压缩
		zip(*iterables)
			可以传入一个或多个可迭代对象，并将各个可迭代对象的对应下标的元素进行压缩，最终返回一个压缩对象
			每个元素都被打包成了元组
			zip对象是一个可迭代对象
			各个可迭代对象，以最短的为准，如果最短的可迭代对象压缩完毕，则zip执行结束（可能造成压缩损失）
5. enumerate():枚举
		enumerate(iterable[, start])
			将可迭代对象的元素和下标进行打包，返回一个完整的对象
			enumerate对象是一个可迭代对象
~~~
